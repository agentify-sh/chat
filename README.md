# Agentify Chat

Agentify Chat lets you talk to a local coding agent from the browser while your computer runs the local Agentify CLI.

This repository is a public placeholder for:

- bug reports
- product feedback
- discussions
- release notes and public-facing documentation

The open source code release is coming. For now, this repository intentionally does not contain backend code or implementation source.

Use the app at https://chat.agentify.sh.

Install the CLI from npm: https://www.npmjs.com/package/@agentify/cli
